# kcptun-gui
GUI for kcptun (https://github.com/xtaci/kcptun).  (.NET framework 4.5)

[![Build Status]][Appveyor]

### 功能

* 可视化参数设置
* 多服务器切换
* 开机启动
* 流量统计（实验性）
